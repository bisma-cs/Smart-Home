# 🔥 Smart Home GUI

A Java Swing-based desktop application that simulates a **Smart Home Controller**.  
This project provides a graphical interface to manage and monitor various smart devices such as lights, fans, thermostats, door locks, CCTV cameras, and smart speakers.

---

## ✨ Features

- **Interactive GUI** built with Java Swing
- **Device Management**:
  - Lights 💡 (toggle ON/OFF)
  - Fan 🌀 (adjust speed 1–3)
  - Thermostat 🌡 (set temperature 10–30°C)
  - Door Lock 🔒 (lock/unlock with confirmation)
  - CCTV 📷 (toggle recording state)
  - Smart Speaker 🔊 (mute/unmute control)
- **Custom Device Settings** dialogs for fan speed, thermostat temperature, and speaker controls
- **Dynamic Status Panel** with HTML rendering for colorful, real-time device status
- **Keyboard Shortcuts**:
  - `Space` → Toggle selected device
  - `Enter` → Open settings for selected device
- **Auto-refresh**: Device status updates every 3 seconds
- **Modern UI Styling** with hover effects, icons, and themed panels
